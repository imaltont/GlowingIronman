glowing-ironman
===============

La koden for MapVelger inn her for å lettere finne den igjen. Kjørbar fil er MapVelger.jar, den må være i samme mappe som "maps" mappen. Dersom du bare skal ha programmet kan du laste ned MapVelger.rar, og pakke det ut en plass, og så kjøre jarfilen derfra (.rar filen inneholder maps mappen og MapVelger.jar).
